## Run Locally

Clone the project

```bash
  git clone https://github.com/kelani34/metabnb
```

Go to the project directory

```bash
  cd metabnb
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Deployment

To deploy this project run

```bash
  npm run build
```
